# No-Play Rules

System ma twardo blokowac markety, ktore nie pasuja do defensywnego edge.

## Globalne blokady

- nie gramy `draw YES`
- `draw NO` jest dozwolone tylko gdy aktualna roznica goli wynosi minimum `2+`
- nie gramy comebackow wymagajacych odwrocenia wyniku
- nie gramy handicapow `minus`, jezeli aktualny wynik jeszcze nie pokrywa linii
- nie gramy `corners`
- nie gramy `anytime goalscorer`
- nie gramy `exact score`
- nie gramy `BTTS`
- nie gramy marketow typu:
  - `next goal`
  - `another goal`
  - `any more goals`
  - `will there be a goal`
  - podobnych rynkow wymagajacych nowego gola

## Glowna zasada

Nie gramy zdarzen, ktore musza dopiero zajsc.

Gramy tylko to, co:

- juz jest blisko spelnienia
- albo moze sie nie wydarzyc
